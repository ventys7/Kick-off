# ⚽️ Kick-off — Football Match Countdown

A lightweight, modern web dashboard to keep track of the next matchday across Europe's top leagues: **Serie A**, **Premier League**, and **La Liga**.

---

### 🔗 Live Version
To use the app or see it in action, please use the official link provided in the repository description.

---

### ✨ Features
* **Live Countdown:** Smart timer (Days/Hours/Minutes) that updates in real-time.
* **Smart Matchdays:** Automatically detects and displays the next upcoming round.
* **Simultaneous Games:** Highlights how many matches are being played at the same time.
* **Modern UI:** Responsive design with glassmorphism effects, blurs, and gradients.
